☕ Roastery Coffee House

Roastery Coffee House is a modern, visually immersive landing page designed to showcase what I can do as a frontend developer. It's not just a coffee shop website — it's a storytelling experience that uses smooth parallax scrolling, immersive zoom transitions, and cinematic animations to take visitors on a journey through the brand's vibrant world.

This project was built to demonstrate my ability to create high-performance, eye-catching, and creative web experiences using modern frontend tools and libraries.

🎨 Features

Storytelling Parallax Scroll - Engaging scroll-driven narrative with layered depth

Immersive Zoom Transitions -Smooth zoom effects that guide users between sections

Snaky Animated Menu - Unique, flowing menu that enhances the browsing experience

Custom Loading Animation - Filling pipe effect — creative preloader that builds anticipation

Performance Optimized - 	Lazy loading, optimized assets, smooth 60fps animations

🛠️ Built With

React.js – UI library

GSAP – ScrollTrigger, timeline animations, parallax effects

Framer Motion – Smooth component transitions and zoom effects

Sass – Advanced styling with nesting, variables, and mixins

CSS3 – Custom layout and responsive design

Vite – Fast build tool and development server

🧠 The Process

I started with a simple idea: a coffee shop website shouldn't just list products — it should tell a story. I wanted to create a digital journey that feels as rich and warm as a good cup of coffee.

First, I designed the loading experience — a filling pipe animation that sets the tone while assets load in the background. It’s small, but it signals quality.

Next came the scroll experience. I used GSAP ScrollTrigger to build a parallax effect where background elements move at different speeds, creating depth as the user scrolls. Then I added immersive zoom animations — as you move between sections, the content scales and reveals new details, like stepping into a different room.

The snaky menu was a creative challenge. I wanted navigation to feel organic, not rigid. Using GSAP and Framer Motion, I created a menu that curves and slides smoothly, matching the flow of the page.

Throughout the build, I focused on performance:

Optimized images (WebP format, lazy loading)

Used will-change and GPU-accelerated properties

Minimized layout shifts and repaints

Kept bundle size lean with code splitting

The result is a site that’s not only beautiful but also fast and smooth — even on mobile devices.

📚 What I Learned

GSAP Mastery – Gained deep understanding of timelines, scroll triggers, and advanced easing functions.

Performance Optimization – Learned how to profile and fix jank, optimize asset loading, and maintain 60fps scrolling.

Framer Motion & React – Combined React’s component model with animation libraries to create reusable, animated UI pieces.

Creative Storytelling – Realized that code can be a medium for narrative, not just functionality.
